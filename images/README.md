## üí° Image Optimization Tips

1. **File Size**: Keep each image under 300KB for fast loading
2. **Format**: Use JPG for photos, PNG only if transparency is needed
3. **Compression**: Use tools like TinyPNG, ImageOptim, or Squoosh to compress
4. **Dimensions**: 
   - Product images: 600px width minimum
   - Hero background: 1920px width
   - Testimonial photos: 200x200px minimum
   - Gallery photos: 600px width minimum
5. **Aspect Ratios**: 
   - Product cards: Square (1:1) or 4:3 works best
   - Hero: 16:9 or wider
   - Testimonials: Square (1:1)

---

## üé® Image Style Guidelines

- **Product Images**: Should be clean, well-lit, showing the pastry clearly. White or neutral background preferred.
- **Hero Background**: Should evoke Yunnan gardens, mountains, or rose fields. Natural, serene feeling.
- **Gift Set**: Should showcase elegant packaging, premium feel.
- **Testimonials**: Professional headshots or friendly customer photos.
- **Gallery**: Real customer photos of pastries, unboxing, or enjoying the product.
